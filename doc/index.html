The core

Overview

GridLayout is a light-weight CSS library to theme and build responsive websites with new CSS components (CSS Grid, CSS Custom properties) and Sass. Two compressed CSS files are provided :

  • css/gridlayout-full.min.css (20ko)
  • css/gridlayout-grid-only.min.css (8ko)

Just import the one you want to use in your project :

Breakpoints

By default, GridLayout uses the same breakpoints as Bootstrap for responsiveness. Some of classes use these class infixes so as not to disturb Bootstrap regular users :

Breakpoint Class infix {breakpoint} Dimensions
X-Small None <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra large xl ≥1200px
Extra extra large xxl ≥1400px

Containers

Containers are usefull to wrap your content. They just apply a padding. Here are the two classes :

  • .container (is a responsive, fixed-width container, meaning its max-width changes at each breakpoint)
  • .container-fluid (for a full-width container)

The grid

Use the .grid class to get a grid composed of 12 columns. All its children are 12 columns width by default.

Columns

The children of the .grid wrapper are 12 columns width by default. You can define the width for each column with these responsive classes :

  • .col-{size}
  • .col-{breakpoint}-{size}

{size} is between 1 and 12.

Row spanning

You can span rows with the .row-{size} and the .row-{breakpoint}-{size} classes. {size} is between 2 and 5.

.col-md-5.row-lg-2
.col-md-7

Offset

You can offset columns by using the .start-{position} and the .start-{breakpoint}-{position} classes. {position} is between 1 and 12.

.col-md-4
.col-md-4.start-md-7

Gutters

You can add gutters to your grid by using the following classes :

  • .g-{size}
  • .gx-{size} for horizontal gutters
  • .gy-{size} for vertical gutters

{size} is between 1 and 5.
