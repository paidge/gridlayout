The helpers

Overview

To facilitate front developement, the library provides some helper classes.

Headings

You can use .h1 to .h6 classes to apply the same responsive font size than headings.

Images

The library provides two responsive classes : .img-fluid and .img-thumbnail. The second extends the first one (max-width : 100% and height : auto).

Display

The libray provides the following classes :

  • .d-{display}
  • .d-{breakpoint}-{display}
  • .m-auto, .mx-auto, my-auto

{display} is a value amongst [none, block, inline-block, flex, grid].

Spacing

The library provides the following classes :

  • .{type}-{size}
  • .{type}{sides}-{size}

{type} is a value amongst [p, m] corresponding to padding and margin.

{size} is a number between 0 and 5.

{sides} is a value amongst [x, y, t, b, r, l] corresponding to [horizontal, vertical, op, bottom, righ, left].

Border-radius

You can use .round-circle and .round-{size} classes to apply a border-radius. {size} is between 0 and 5.

.round-circle
.round-4

Tables

Add .table-striped class to your table in order to highlight even and odd lines. You can also wrap your table with the .table-responsive class (in a .grid).
