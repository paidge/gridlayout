CSS components

You can customize these colors directly in CSS by using the appropriate CSS component --{color}.

Text

The --default-font-color CSS component defines the font color for the html and body elements and is usable only for :root.

Use the text-{color} classes to apply the appropriate color to the text.

.text-theme1

Borders

Use the .border and the .border-{size} classes to apply a border whose the color is defined by the --border-color CSS component.

Use the border-{color} and the border-{color}-{size} classes to apply a border with the appropriate color.

{size} is between 2 and 5.

.border-theme4-3

Background

You can use the .bg-{color} classes or the --bg-color CSS component.

.bg-theme2

Themes

The .{color} classes are equivalent to the .text-{color} classes. In addition, they overload the --primary, the --inv-primary, the --focus-shadow and the --border-color CSS components with the appropriate colors. They are usefull for wrappers.

.border-1.text-primary
